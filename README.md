# Gabarita AI Frontend

Frontend application for Gabarita AI - an intelligent educational platform.

## Technologies Used

- React 19
- Next.js 15
- TypeScript
- Tailwind CSS
- Radix UI Components
- Firebase
- Framer Motion

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/techiaemp-netizen/gabarita-ai-frontend.git
cd gabarita-ai-frontend
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint
- `npm test` - Run tests
